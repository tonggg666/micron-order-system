📦 Micron Project 订货管理系统
这是一个基于纯 HTML、CSS 和 JavaScript 构建的简易、高效的订货单生成工具，专为 Micron 项目的物料采购流程设计。它实现了物品信息持久化存储（LocalStorage），并支持一键生成符合要求的 Word 格式订货单。

✨ 核心功能亮点
纯前端运行： 单个 test30.html 文件即可运行，无需服务器或复杂的数据库。

本地物品库：

支持录入物品名称、单位、规格、文本备注。

支持为物品上传 详情照片 (Base64 存储在本地浏览器)。

提供 弹窗编辑 功能，方便对已录入的物品进行修改、替换图片或删除。

智能订货表单： 在实际订货页面输入物品名称时，提供联想输入功能，自动填充单位、规格和详情图片。

一键生成 Word 订货单：

生成标准 .doc 格式文件，兼容 Microsoft Word。

【优化】 文档采用 A4 横向 (Landscape) 格式，排版更舒展。

【优化】 物品详情图片被放大并固定尺寸（如 180px 宽），确保图片清晰可见，每页可容纳约 3-4 行项目。

数据持久化： 物品库数据存储在浏览器的 LocalStorage 中，刷新或关闭浏览器后数据不会丢失。

⚙️ 技术栈
HTML5: 结构

CSS3: 简洁、淡绿色的现代化 UI 样式

纯 JavaScript (ES6): 核心逻辑、Word 文件生成、数据管理（LocalStorage）

🚀 如何使用 (本地运行)
此系统设计为在任何现代浏览器中本地运行，无需任何配置。

克隆或下载仓库：

Bash

git clone https://github.com/YOUR_USERNAME/Micron-Order-System.git
运行文件： 找到 test30.html 文件，用您电脑上的任意浏览器（如 Chrome, Edge, Firefox）直接打开即可。

💡 使用指南
设置概览信息： 填写订货日期、送货日期等必填信息。

录入物品到系统 (物品库)：

在 📥 物品录入系统 模块中，输入新的物品信息，并上传详情照片（可选）。

点击 “录入新物品到系统” 按钮保存。

编辑物品库： 点击 “查看/编辑已录入物品列表” 按钮，您可以在弹窗中直接修改名称、单位，或上传新的图片替换旧图片。

填写订货单：

在 🛒 实际订货页面，输入物品名称时，会弹出联想列表。选择后，单位、规格和详情照片将自动填充。

填写 QTY (数量)。

生成订货单： 点击底部绿色的 “一键生成订货单 WORD (.doc)” 按钮。

Word 提示： 下载的 Word 文件如果提示“受保护的视图”，请务必点击 [启用编辑]，图片内容才能正常加载显示。
